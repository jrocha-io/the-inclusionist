# LM-GM Map — Learning Mechanics ↔ Game Mechanics

The bridge between the game layer and the curriculum layer (Arnab et al. 2015, *BJET* 46(2)). It names **which game
mechanic can carry which learning mechanic** — i.e. the **strategic points where a classroom activity is inserted** and
rewarded gently after the child wins (ADR-0006; the Dev's core method — build the *lúdico* first, then insert learning).

- **Game mechanics** come from [`genre-catalog.md`](genre-catalog.md).
- **Learning mechanics / objectives** come from [`../educational/Learning-Objectives.md`](../educational/Learning-Objectives.md)
  (BNCC + Bloom + cognitive skill).
- Each row is a candidate **insertion point**; a chosen pairing becomes a per-game one-sheet + an issue.

## Map (seed — grows as games are studied)

| Learning mechanic | Fits which game mechanic (genre #) | Insertion note |
|---|---|---|
| Phoneme→grapheme matching | Memory/Concentration (7), Match-3 (4) | pair sound ↔ letter; reward on match |
| Classification / sorting | Color-sort (4), Cozinha quality-control (31), recycling-sort (14) | sort by an attribute the objective targets |
| Sequencing / ordering | Simon (7), Rhythm (17), assembly line (31) | order syllables / steps / events |
| Spelling / lexicon | Typing (18), Word puzzles (5), Hangman (5) | build/recall words |
| Quantity / number sense | Quiz (20), Physics-aim (6), Plinko/probability (13→ probability learning) | count / estimate / compare |
| Recall / retrieval | Quiz (20) as a reward gate after any lúdico | the pattern used so far (math/literacy minigames) |
| Spatial reasoning | Maze (34), Sokoban (4), Isometric (28) | navigate / plan a path |
| Cause & effect / systems | Sandbox sim (26), Tower defense (16) | observe a rule, predict an outcome |

## Method (per game)

1. Build the genre as a **lúdico** (fun first).
2. Find, in its **beat chart**, the strategic points to insert the activity.
3. Pick the learning objective (educational/) whose learning mechanic matches the game mechanic (this table).
4. Insert the activity; reward **gently** on win (no dopamine spikes — ADR-0006).
5. Record the pairing here + a per-game one-sheet; open an issue.
